# 安装插件

## MCDR 快捷安装
   - 在已经开启的 MCDReforged MC 服务端输入以下指令可快捷安装插件及前置：
   ```text
   !!MCDR plugin install gugubot
   ```
## 手动下载安装
   1. 下载插件依赖  
   ```bash
   pip install websocket-client
   ```
   2. 前往插件 [MCDR 插件仓库](https://mcdreforged.com/zh-CN/plugin/cq_qq_api) 下载最新版插件并放入 MCDR 根目录下的 `plugins` 文件夹:
